# :mag: GitHub Repository Analyzer

### Analyze any GitHub repo and generate comprehensive AI-ready reports

---

## :dart: What Does This Do?

This tool **analyzes any GitHub repository** and generates a **single comprehensive file** containing:

| What | Description |
|------|-------------|
| :clipboard: **Project Info** | Name, description, stars, forks, license, topics |
| :evergreen_tree: **File Tree** | Complete visual directory structure |
| :bar_chart: **Statistics** | Lines of code, languages breakdown, file sizes |
| :page_facing_up: **ALL Source Code** | Complete content of every code file |
| :busts_in_silhouette: **Contributors** | Who contributed and how much |
| :memo: **Commit History** | Latest changes |
| :robot: **AI Prompt Template** | Ready-to-use instructions for AI |

### :bulb: Why Use This?

When you want AI (ChatGPT, Claude, Gemini) to **modify or build upon a project**, you need to share the **complete codebase**. Manually copying files is tedious. This tool does it **automatically** and generates a perfectly formatted report!

### :repeat: Workflow

```
1. Run this tool on any GitHub repo
2. Get a complete report with ALL code
3. Send the report to AI
4. Ask AI to modify/improve/build upon it
5. Get back modified code!
```

---

## :star2: Features

| Feature | Description |
|---------|-------------|
| :unlock: **Public Repos** | Analyze any public repo without a token |
| :lock: **Private Repos** | Access private repos with your token |
| :mag: **Browse Repos** | See numbered list of your repos, select by number |
| :bust_in_silhouette: **Auto-detect User** | Automatically gets your info from token |
| :herb: **Branch Support** | Choose which branch to analyze |
| :page_facing_up: **TXT Export** | Best for sending to AI |
| :closed_book: **PDF Export** | Formatted documentation |
| :blue_book: **Word Export** | Editable .docx report |
| :package: **ZIP Export** | All formats compressed |
| :arrow_down: **Auto-download** | Files download automatically in Colab |

---

## :key: Creating a GitHub Token

1. Go to [github.com/settings/tokens](https://github.com/settings/tokens)
2. Click **Generate new token (classic)**
3. Select scope: :white_check_mark: `repo` (Full control)
4. Copy the token

---

## :robot: Using with AI

1. Generate the **TXT report**
2. Open it and **copy all content**
3. Paste to **ChatGPT / Claude / Gemini**
4. Add your request:

```
Here is my complete project with all source code.
I want to:
1. Add a new feature that does X
2. Fix the bug in file Y
3. Improve performance of Z
Please provide the modified code.
```
